Инструкция

Для начала работы нужно выбрать xml файл для визуализации,
нажав кнопку Открыть.

Внесенные изменения можно сохранить в исходный файл кнопкой
Сохранить.

-WASD двигают камеру вперед-назад-влево-вправо
-Q двигает камеру вверх
-Z двигает камеру вниз

-Щелчок левой кнопкой мыши по хосту выделяет его. Выделенный
хост отображается синим цветом.
-Shift+клик по хостам позволяет выделять больше одного хоста
-Ctrl+клик по выделенному хосту снимает выделение
-Alt+клик по хосту создает линки к этому хосту от всех выделенных хостов

-Щелчок по линку выделяет этот линк. Выделенные линки отображаются синим
-Shift+клик по линкам позволяет выделять больше одного линка
-Ctrl+клик по выделенному линку снимает выделение

-Delete удаляет все выделенные линки
