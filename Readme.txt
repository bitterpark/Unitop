Инструкция

Для начала работы нужно выбрать xml файл для визуализации,
нажав кнопку Open

Созданные изменения можно сохранить в исходный файл кнопкой
Save.

-WASD двигают камеру вперед-назад-влево-вправо
-Q двигает камеру вверх
-Z двигает камеру вниз
-Щелчок левой кнопкой мыши по хосту выделяет его. Выделенный
хост отображается зеленым цветом.
-Shift+клик по хостам позволяет выделять больше одного хоста
-Ctrl+клик по выделенному хосту снимает выделение
